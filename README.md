# 🏠 BWA Cozy App

Aplikasi **BWA Cozy** adalah project Flutter untuk mencari kos kosan yang nyaman (cozy spaces) dengan desain modern dan user-friendly. Project ini dibuat sebagai bagian dari pembelajaran Flutter sekaligus portofolio pengembangan mobile app.

---

## ✨ Features

* 🔍 Menampilkan daftar kosa kosan
* 🖼️ Galeri foto horizontal pada detail space
* ⭐ Sistem rating
* 🧭 Navigasi antar halaman (Home → Detail)
* 🎨 UI modern dengan clean design

## 🛠️ Tech Stack

* **Flutter**
* **Dart**
* **Provider (State Management)**
* **url_launcher**
* **http**
* **REST API / Dummy Data**

---

## 📂 Project Structure

```bash
lib/
├── models/        # Data model (space, city, dll)
├── pages/         # Halaman UI (home, detail, splash)
├── providers/     # State management
├── widgets/       # Komponen reusable (card, rating, dll)
├── theme.dart     # Styling aplikasi
```

---

## 🚀 Getting Started

1. Clone repository ini:

```bash
git clone https://github.com/USERNAME/bwa_cozy.git
```

2. Masuk ke folder project:

```bash
cd bwa_cozy
```

3. Install dependencies:

```bash
flutter pub get
```

4. Jalankan aplikasi:

```bash
flutter run
```
